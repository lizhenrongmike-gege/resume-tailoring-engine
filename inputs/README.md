# Inputs

Place your resume files here before triggering the skill.

| File | Purpose | Used By |
|------|---------|---------|
| `resume_bank.docx` | **Content source.** Comprehensive file with ALL past experiences. Often >1 page. | Phase 0b (Career Ledger) |
| `resume_master.docx` | Optional legacy file. Kept only for historical reference. | Not required by the current formatting workflow |

The **Job Description (JD)** is not a file — paste it directly into the chat when prompted.

## Notes

- The Resume Bank should contain every role, project, and skill you've ever had — more is better. The skill selects what's relevant per JD.
- Formatting no longer comes from `resume_master.docx`; the skill uses a fixed ATS-safe profile from `references/format_detection.md`.
- If `resume_bank.docx` is missing, the agent will ask you to provide it.
